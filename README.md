# 🖨️ Matrix Graphics

A modern, high-performance, and visually stunning web application for **Matrix Graphics**, a premium print studio located in Avissawella, Sri Lanka.

This project is built with **React**, **Vite**, and **Tailwind CSS**, featuring a fully responsive design, seamless dark/light mode integration, and premium UI aesthetics.

---

## ✨ Key Features

- **Dynamic Theme System**: Flawless transition between Light Mode (sleek slate tones) and Dark Mode (deep navy glassmorphism).
- **Responsive Layout**: Optimized for all screen sizes, from ultra-wide desktops to compact mobile devices.
- **Premium Aesthetics**: High-end styling including glassmorphism, soft dropshadows, and custom SVG styling.
- **Interactive Animations**: Custom scroll-reveal animations using Intersection Observer and smooth CSS cubic-bezier easing.
- **Direct Communication**: Integrated floating WhatsApp action button and direct inquiry routing.
- **Live Location**: Embedded interactive Google Maps iframe for easy physical shop navigation.

## 🛠️ Technology Stack

- **Framework**: React 19 + Vite
- **Routing**: React Router DOM v6
- **Styling**: Tailwind CSS + Vanilla CSS (for custom variables and complex animations)
- **Icons & Assets**: Custom optimized SVGs and heavily optimized WebP images

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Shahidh115/matrix-graphics-site.git
   ```
2. Navigate to the project directory:
   ```bash
   cd my-react-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server with Hot Module Replacement (HMR):
```bash
npm run dev
```

### Building for Production

To create an optimized production build:
```bash
npm run build
```
The bundled files will be generated in the `dist` directory.

## 📁 Project Structure

- `/src/assets` - Static assets including images, logos, and SVGs.
- `/src/components` - Reusable UI components (Navbar, Footer, ServiceCard, WhatsAppButton, etc).
- `/src/pages` - Main page views (Home, About, Services, Gallery, FAQ, Contact).
- `/src/hooks` - Custom React hooks (e.g., `useScrollReveal`).
- `/src/data` - Local static data for rendering dynamic content.
- `index.css` - Global CSS variables and custom styles.

## 🎨 Design Philosophy

The aesthetic of Matrix Graphics is built around a **Premium Print Studio** vibe. The interface utilizes a highly tactile approach, making digital elements feel physical—cards lift on hover, shadows are deep and rich, and the color palette relies on high-contrast pairings (Gold/Yellow on Navy/Slate) to guide the user's eye naturally.
